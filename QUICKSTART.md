# 快速开始指南 - 垃圾应用清理器

## 🚀 5分钟快速上手

### 第一步：安装应用
1. 使用 Android Studio 打开项目
2. 连接手机或启动模拟器
3. 点击运行按钮安装到设备

### 第二步：基本操作流程

#### 场景1：清理不需要的应用
```
1. 打开应用 → 自动显示所有已安装的第三方应用
2. 浏览列表，找到不想卸载的重要应用（如微信、支付宝等）
3. 点击这些应用右侧的 🔒 按钮 → 加入白名单
4. 勾选其他想要卸载的应用
5. 点击底部红色"一键卸载全部"按钮
6. 确认卸载操作
7. 完成！系统会逐个打开卸载页面
```

#### 场景2：批量管理白名单
```
1. 长期使用的应用加入白名单（只需操作一次）
2. 点击"保存白名单"按钮保存配置
3. 下次使用时，白名单自动加载
4. 新安装的应用会出现在列表中，可继续添加到白名单
```

### 第三步：界面说明

#### 顶部区域
- **标题**：垃圾应用清理器
- **统计信息**：显示总应用数和白名单数量
- **操作按钮**：
  - 全选：选中所有非白名单应用
  - 取消全选：取消所有选择
  - 保存白名单：保存当前配置

#### 中间列表区域
每个应用项包含：
- ☑️ 复选框：选择是否卸载
- 📱 应用图标和名称
- 📦 包名（灰色小字）
- 🔓/🔒 白名单切换按钮

#### 底部操作区
- **一键卸载全部按钮**（红色）：执行批量卸载
- **➕ 浮动按钮**（绿色）：快速添加白名单

## 💡 使用技巧

### 技巧1：首次设置白名单建议将以下类型应用加入白名单：
- ✅ 社交软件：微信、QQ、钉钉、微博等
- ✅ 支付工具：支付宝、银行APP等
- ✅ 系统必要：应用商店、电话、短信等
- ✅ 工作相关：邮箱、办公软件等
- ✅ 常用工具：浏览器、地图、相机等

### 技巧2：定期清理流程
```
每周/每月执行一次：
1. 打开应用查看新安装的应用
2. 将常用的加入白名单
3. 选择不用的应用
4. 一键清理
5. 保存更新后的白名单
```

### 技巧3：识别垃圾应用的技巧
- 很久没打开过的应用
- 安装后只使用过一两次的应用
- 功能重复的应用（如多个计算器）
- 占用空间大但很少使用的游戏
- 不再需要的临时工具类应用

## ⚠️ 注意事项

### 卸载前必读
1. **备份数据**：卸载前确认应用内重要数据已备份
2. **检查关联**：某些应用卸载可能导致其他功能异常
3. **账户绑定**：检查是否有账户或订阅服务绑定
4. **离线内容**：部分应用有离线下载内容，卸载会丢失

### 安全建议
- ❌ 不要卸载不认识的重要系统组件
- ❌ 不要卸载正在运行的关键服务应用
- ❌ 批量卸载前先手动测试单个卸载
- ✅ 优先卸载明确知道不再需要的应用
- ✅ 保持白名单及时更新

## 🔧 故障排除

### 问题1：应用没有出现在列表中
**原因**：可能是系统应用  
**解决**：当前版本只显示第三方应用，系统应用不会列出（更安全）

### 问题2：无法卸载某个应用
**原因**：该应用是系统级应用或需要特殊权限  
**解决**：这是正常保护机制，系统核心应用不允许普通方式卸载

### 问题3：白名单丢失
**原因**：清除应用数据会导致白名单重置  
**解决**：定期点击"保存白名单"，或避免清除本应用的数据

### 问题4：卸载后应用还在
**原因**：用户在系统卸载页面取消了卸载操作  
**解决**：需要在弹出的卸载对话框中确认删除

## 📊 性能说明

- **扫描速度**：通常 1-3 秒完成（取决于安装应用数量）
- **内存占用**：约 20-50MB（运行时）
- **存储占用**：安装包约 5MB，白名单数据 < 10KB
- **电池影响**：仅在扫描和操作时消耗电量，后台无消耗
